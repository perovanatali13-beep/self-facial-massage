# CLAUDE.md

Указания для Claude Code по этому репозиторию. Полное руководство по проекту
(стек, структура, модель данных, авторизация, переменные окружения, деплой и нюансы)
ведётся в едином файле — он импортируется ниже, чтобы не дублировать содержимое.

@AGENTS.md

## Кратко самое важное

- БД (Supabase) — источник правды в рантайме; `data/*.json` — только сид. Правка
  документов `content`/`course` в Supabase применяется на сайте сразу, без передеплоя.
- `git push` в `main` автоматически деплоит прод (Vercel git integration).
- Секреты только в env (`.env.local` / Vercel), никогда в коде и без `NEXT_PUBLIC_`.
- Язык интерфейса — русский.
